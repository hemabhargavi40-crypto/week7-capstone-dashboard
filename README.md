# Student Academic Insights Dashboard

## Overview
This project analyzes student habits like study hours, attendance, and sleep.

## Features
- Data Cleaning
- Visualization (Scatter, Heatmap, Histogram)
- Machine Learning Model (Logistic Regression)

## Results
- Model predicts student performance categories
- Accuracy shows good performance

## Insights
- More study hours → better grades
- High attendance → better results
- Balanced sleep improves performance
